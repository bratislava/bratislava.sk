# NumericalList

NumericalList

## Props

NumericalList accepts custom Props that are as follows:

- <b>title</b>: String that is the title of the numerical list
- <b>items</b>: Array of items that are displayed in the numerical list
- <b>buttonText</b>: String that is displayed inside button
- <b>buttonLink</b>: String that represents the link on which the button links us to
- <b>variant</b>: String that controls the numerical list design. Can be one of "basic", "combined" or "roadmap".

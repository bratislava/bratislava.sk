# Radio

Styled Radio, that is used as a radio throughout all Bratislava web apps.

## Props

Radio accepts all default HTMLInputProps in addition to custom Props that are as follows:

- <b>key</b>: String that gives key to the radio in document
- <b>title</b>: String that is the inner text content of the radio

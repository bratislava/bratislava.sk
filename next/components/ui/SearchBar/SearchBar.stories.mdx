# SearchBar

Styled SearchBar, that is used as a searchBar throughout all Bratislava web apps.

## Props

Radio accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>value</b>: inner text value of the searchBar
- <b>onChange</b>: Value ChangeHandler function

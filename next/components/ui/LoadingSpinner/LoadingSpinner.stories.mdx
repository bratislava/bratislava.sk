# LoadingSpinner

Styled LoadingSpinner, that is used as a primary LoadingIndicator throughout all Bratislava web apps.

## Props

LoadingSpinner accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>size</b>: determines the size of the LoadingSpinner
- <b>text</b>: text, that is attached to the LoadingSpinner

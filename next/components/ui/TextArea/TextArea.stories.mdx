# TextArea

Styled TextArea, that is used as a TextArea throughout all Bratislava web apps.

## Props

TextArea accepts all default HTMLTextAreaProps in addition to custom Props that are as follows:

- <b>hasError</b>: Boolean value, that indicates, that the textarea should have
  error styling (red border)

# FooterCard

Styled Card, that consists of the Top ReactNode(s) and a Footer ReactNode

## Props

FooterCard accepts all default HTMLDivProps in addition to custom Props that are as follows:

- <b>footer</b>: ReactNode, that is positioned under the children of this component

# TimeCardMobile

Styled TimeCard, that is used 'as' a TimeCard 'for' Mobile Devices in OrderSteps of the app Plavarne.

## Props

TimeCardMobile accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>primaryContent</b>: 'Left part of the TimeCard' content
- <b>secondaryContent</b>: 'Right part of the TimeCard' content
- <b>onClick</b>: ClickHandler function
- <b>isActive</b>: Boolean to indicate the active state
- <b>tabIndex</b>: tabIndex 'for' the accessibility features

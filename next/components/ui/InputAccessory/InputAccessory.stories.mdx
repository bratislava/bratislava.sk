# InputAccessory

StyledDiv Element, that consists of Icon and an input element.

## Props

InputAccessory accepts custom Props that are as follows:

- <b>className</b>: modifieds or extends the styles for the component
- <b>hasError</b>: Boolean value, that indicates, that this component should have error styling (red border)
- <b>children</b>: React.ReactNode(s), that behaves as a child(ren) of this component
- <b>icon</b>: ReactNode(e.g. SVG) that will behave as an icon within this COmponent
- <b>iconPosition</b>: position of the given icon

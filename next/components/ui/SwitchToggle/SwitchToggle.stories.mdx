# SwitchToggle

Styled SwitchToggle, that is used as a SwitchToggle with SwitchToggleTabs as its tabs.

## Props

SwitchToggle accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>titleLeft</b>: clickable string, that is positioned on the left of switch
- <b>titleRight</b>: clickable string, that is positioned on the right of switch
- <b>value</b>: 'left' or 'right'
- <b>onValueChange</b>: SwitchValue Change Handler Function

export const Template = (args) => {
  return <SwitchToggle className={args.className} onValueChange={() => {}} />;
};

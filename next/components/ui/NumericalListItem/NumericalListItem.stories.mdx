# NumericalListItem

NumericalListItem

## Props

NumericalListItem accepts custom Props that are as follows:

- <b>index</b>: By one number smaller than the number displayed left to text
- <b>item</b>: Object of which content is displayed
- <b>variant</b>: String that controls the numerical list item design. Can be
  one of "basic", "combined" or "roadmap".

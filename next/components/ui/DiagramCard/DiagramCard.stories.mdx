# DiagramCard

Styled Card, that consists of Image and description.

## Props

- <b>className</b>: modifieds or extends the styles for the component
- <b>imgSrc</b>: StaticImageData, that serves as a source of the next/image
- <b>imgWidth</b>: Width of StaticImageData, that serves as a source of the
  next/image
- <b>imgHeight</b>: Height of StaticImageData, that serves as a source of the
  next/image
- <b>text</b>: String, that is positioned under the image
- <b>index</b>: Number, that is positioned on the top of the Component

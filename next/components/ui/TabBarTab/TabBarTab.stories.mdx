# TabBarTab

Styled TabBarTab, that is used as a tab of TabBar Component

## Props

TabBarTab accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>title</b>: String, that is the title of the Tab Component
- <b>onClick</b>: tab ClickHandler Function
- <b>isActive</b>: Boolean, that indicates an active state of the Tab

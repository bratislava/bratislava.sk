# Footer

Footer

## Props

Footer accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>FBLink</b>: link to FaceBook
- <b>IGLink</b>: link to InstaGram
- <b>YTLink</b>: link to YouTube
- <b>address</b>: address as string
- <b>email</b>: email as string
- <b>phone</b>: phone number as string
- <b>sections</b>: sections of footer
- <b>accessibility</b>: accessibility as string
- <b>copyright</b>: copyright as string
- <b>lang</b>: language as string

# RadioGroup

RadioGroup, that wraps radio buttons throughout all Bratislava apps.

## Props

RadioGroup accepts custom Props that are as follows:

- <b>className</b>: className for the radio group
- <b>radioClassName</b>: className for the radio button
- <b>options: T[]</b>: options(radio buttons) for the radiogroup
- <b>onChange</b>: function, accepts two parameters - selected option and if is
  checked
- <b>value</b>: String or T to set the 'checked' state of the radio

export const Template = (args) => {
  const [option, setOption] = React.useState('0');
  return (
    <RadioGroup
      className={args.className}
      radioClassName={args.radioClassName}
      options={args.options}
      onChange={(opt) => setOption(opt.key)}
      value={option}
    />
  );
};

# DateSelectShortcuts

DateSelectShortcuts

## Props

DateSelectShortcuts accepts custom Props that are as follows:

- <b>value</b>: string of the current local time evaluated using ISO chronology
- <b>onClick</b>: function, accepts as parameter selected day as date in string
  format

export const Template = (args) => {
  const [value, setValue] = React.useState(LocalDate.now().toJSON());
  return (
    <DateSelectShortcuts
      className={args.className}
      value={value}
      onClick={(d) => setValue(d)}
    />
  );
};

# InfoCard

InfoCard

## Props

InfoCard accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>imageSrc</b>: image of the component
- <b>imageWidth</b>: width of the image
- <b>imageHeight</b>: height of the image
- <b>primaryTitle</b>: primary title of the component
- <b>secondaryTitle</b>: secondary title of the component
- <b>description</b>: description of the component
- <b>children</b>: additional component of the component

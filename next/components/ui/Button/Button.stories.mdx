# Button

Styled Button, that is used as a primary and secondary button throughout all Bratislava web apps.

## Props

Button accepts all default HTMLButtonProps in addition to custom Props that are as follows:

- <b>variant</b>: changes the background and text color of the button
- <b>icon</b>: ReactNode that will behave as an icon within the button
- <b>hoverIcon</b>: ReactNode that will behave as a hoverIcon on hover effect of the button if omitted, icon will be
  used when present
- <b>iconPosition</b>: position of the given icon and hoverIcon
- <b>iconClassName</b>: classname of the icons (mostly to adjust padding)

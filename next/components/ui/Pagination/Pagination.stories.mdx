## Props

Pagination accepts custom Props that are as follows:

- <b>currentPage</b>: currentPage
- <b>itemsPerPage</b>: number of items par page
- <b>pageHandler</b>: function which explains the behaviour per page change
- <b>totalCount</b>: number of items

export const Template = (args) => {
  return <Pagination itemsPerPage={args.itemsPerPage} totalCount={args.totalCount} />
}

# TicketTypeCards

TicketTypeCards

- <b>className</b>: modifies or extends the styles for the component
- <b>key</b>: key for ticket types
- <b>title</b>: title for ticket types
- <b>ticketTypes</b>: array for ticket types. Each ticket type has name of type, title, price and optional infotip.

## Props

Field accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>children</b>: main content of the component
- <b>id</b>: id of the component
- <b>title</b>: title of the component
- <b>error</b>: error text of the component

# Link

Styled Link, that is used as a default Link throughout all Bratislava web apps.

## Props

Link accepts custom Props that are as follows:

- <b>icon</b>: ReactNode that will behave as an icon within the button
- <b>hoverIcon</b>: ReactNode that will behave as a hoverIcon on hover effect of the button
- <b>iconPosition</b>: position of the given icon and hoverIcon
- <b>iconClassName</b>: classname of the icons (mostly to adjust padding)
- <b>className</b>: modifies or extends the styles for the component
- <b>href</b>: next router path
- <b>children</b>: ReactNode(s), that behaves as a child(ren) of thi Component

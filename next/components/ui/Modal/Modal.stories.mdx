# Modal

Modal

Modal accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>isOpen</b>: if true, modal is opened, otherwise, is closed
- <b>children</b>: content of modal
- <b>closeButton</b>: content of closeButton
- <b>footer</b>: optional footer for modal

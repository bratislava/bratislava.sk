# Tooltip

Styled Tooltip, that is used as a tooltip throughout all Bratislava web apps.

## Props

Tooltip accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>content</b>: String, that is the content of the tooltip's help message
- <b>target</b>: ReactNode, that can be used as the replacement for the default '?' target

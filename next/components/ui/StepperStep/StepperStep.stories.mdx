# StepperStep

Styled Circle Button, that is used 'as' a Step 'for' our Stepper Component in OrderSteps.

## Props

StepperStep accepts custom Props that are as follows:

- <b>className</b>: className for the StepperStep
- <b>title</b>: String, that is the title of the StepperStep (positioned under the CircleButton)
- <b>number</b>: Number, that is shown inside the Component when not checked
- <b>onClick</b>: StepperStep ClickHandler Function
- <b>active</b> Boolean, that indicates the active state of the Component
- <b>checked</b>: Boolean, that indicates the checked state of the Component

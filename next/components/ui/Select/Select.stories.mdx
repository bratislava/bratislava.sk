# SelectField

Styled select, that is used as a default select throughout all Bratislava web apps.

## Props

CheckBox accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>selectClassName</b>: className for the SelectField Component
- <b>options</b>: options for the select of generic type T
- <b>onChange</b>: function, accepts as parameter selected option as object
- <b>value</b>: selected option's value
- <b>id</b>: identifier of the SelectField Component
- <b>hasError</b>: Boolean value, that indicates, that the SelectField should have error styling (red border)
- <b>hasIcon</b>: Boolean value, that indicates, that the SelectField has an icon (icon is hardcoded ChevronDown positioned
  right of the value)

export const Template = (args) => {
  const [value, setValue] = React.useState('0')
  return (
    <SelectField
      className={args.className}
      options={args.options}
      hasError={args.hasError}
      hasIcon={args.hasIcon}
      value={value}
      onChange={(d) => setValue(d)}
    />
  )
}

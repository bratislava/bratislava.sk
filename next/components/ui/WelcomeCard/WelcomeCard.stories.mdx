# WelcomeCard

Styled Card Component, that is used as a welcome card in WelcomeScreens.

## Props

WelcomeCard accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>image</b>: ReactNode that is positioned on the top of title
- <b>title</b>: String that is the title of the WelcomeCard
- <b>path</b>: String that is the path for the 'href' prop of the "next/link"

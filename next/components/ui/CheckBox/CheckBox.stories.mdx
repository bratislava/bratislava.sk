# CheckBox

Styled Button, that is used as a checkbox throughout all Bratislava web apps.

## Props

CheckBox accepts all default HTMLInputProps in addition to custom Props that are as follows:

- <b>id</b>: String that identifies the checkbox in document (important for right labeling)
- <b>content</b>: ReactNode that will be the content of the message, that is attached to checkbox

# TabBar

Styled TabBar, that is used as a tabbar with TabBarTabs as its tabs.

## Props

TabBar accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>tabs</b>: tabs, that the TabBar displays
- <b>activeTab</b>: tab that is currently active
- <b>onSelect</b>: function, accepts as parameter selected tab as object

export const Template = (args) => {
  const [activeTab, setActiveTab] = React.useState('Tab1')
  return (
    <TabBar className={args.className} tabs={args.tabs} activeTab={activeTab} onSelect={(d) => setActiveTab(d.key)} />
  )
}

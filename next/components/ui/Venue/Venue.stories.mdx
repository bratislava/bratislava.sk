# Venue

// Styled Venue, that is used as a default Venue throughout all Bratislava web apps.

## Props

Venue accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>id</b>: ID of the Space (navigation to detail and reservation based on ID)
- <b>imageSrc</b>: Source of the Image (of the Space)
- <b>title</b>: title of the component
- <b>description</b>: description of Space Rental
- <b>buttonTitle</b>: text in button of the component
- <b>linkTitle</b>: text in link of the component

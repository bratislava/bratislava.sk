# NumberSwitcher

NumberSwitcher

## Props

NumberSwitcher accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>id</b>: ID of the element
- <b>value</b>: Value is typeof Number
- <b>hasError</b>: Red Border if this field has hasError
- <b>onChange</b>: Value ChangeHandler function

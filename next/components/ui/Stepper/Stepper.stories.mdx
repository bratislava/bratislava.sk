# Stepper

Component, that is used 'as' a Step Indicator 'for' large screen devices in OrderSteps.

## Props

Stepper accepts custom Props that are as follows:

- <b>steps</b>: All the steps, that are to be displayed by Stepper
- <b>currentStep</b>: Number, that indicates the current active step
- <b>className</b>: className for the Stepper
